# Meine Lernreise (My Learning Journey)

Willkommen in meinem Repository! Hier dokumentiere ich meine Fortschritte in der Webentwicklung. Ich lerne Schritt für Schritt, wie man moderne Webseiten erstellt und teile hier meine Übungen und kleinen Projekte.

## Technologien & Tools
Aktuell konzentriere ich mich auf:
* **HTML5:** Strukturierung von Webinhalten.
* **CSS3:** Styling, Layouts und Animationen (z.B. Button-Designs, Hover-Effekte).
* **Git & GitHub:** Versionskontrolle und Dokumentation meines Codes.

## Inhalt dieses Repositories
In diesem Repo findest du meine täglichen Übungen, zum Beispiel:
* `button.html`: Design von interaktiven Buttons (z.B. Twitter/X Style).
* `website.html`: Grundlegende Layout-Strukturen.
* `Übung.html`: Verschiedene CSS-Experimente.

- [ ] Erstellung eines persönlichen Portfolios.
